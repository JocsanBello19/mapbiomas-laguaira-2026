# mapbiomas-laguaira-2026
Código del artículo científico (Premio MapBiomas Venezuela 2026) sobre vulnerabilidad histórica e impacto post-sísmico en La Guaira (1985-2026). Integra MapBiomas Col. 3, radar Sentinel-1 en GEE y el Índice IVTH. Incluye modelado predictivo multi-escenario en Python (Cadenas de Markov + Random Forest) para restauración resiliente al 2040.
